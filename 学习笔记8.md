# cookie
  - ### 创建cookie对象
  ```java
   //设置response的编码
        response.setContentType("text/html;charset=UTF-8");

        //创建Cookie对象，指定名称和值
        Cookie cookie = new Cookie("username", "zhongfucheng");
        
        //向浏览器给一个Cookie
        response.addCookie(cookie);

        response.getWriter().write("我已经向浏览器发送了一个Cookie");


  ```
